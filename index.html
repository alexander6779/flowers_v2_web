Tipos de Flores

Clasificación de flores usando la cámara web utilizando Tensorflow.js, para este caso solo clasificamos 5 tipos de flores: margaritas,dientes de león,rosas,girasoles y tulipanes.

Indicaciones: aceptar que el navegador pueda usar la cámara, ajustar la cámara al objetivo en este caso flor y pulsar el botón de obtener resultados, dicho resultado aparecerá debajo del botón.
